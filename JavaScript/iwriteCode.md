1️⃣ Electricity Bill Calculation
A household is charged based on units consumed
First 100 units → ₹5 per unit
Next 100 units → ₹7 per unit
Above 200 units → ₹10 per unit
Write a JavaScript program to calculate the total electricity bill using arithmetic and comparison operators.

A cinema charges prices based on:
Age < 12 → Child ticket ₹150
Age 12–60 → Adult ticket ₹250
Age > 60 → Senior citizen ticket ₹180
Plus:
If the user books online, apply 5% discount (use assignment and logical operators).
Write a JS program to calculate final ticket price.

18.5–24.9 → Normal
25–29.9 → Overweight
≥30 → Obese
